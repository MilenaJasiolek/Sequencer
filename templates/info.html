Information about the program Sequencer

Mój projekt to program zrealizowany w ramach zaliczenia przedmiotu "Programowanie w języku Python z elementami analizy danych i bioinformatyki – rozszerzenie" napisany w języku Python z użyciem biblioteki Django oraz Pandas, który będzie renderował w przeglądarce proste podstrony (wystylowane w CSS) zawierające tabele wygenerowane przy użyciu biblioteki Pandas wykorzystując różne informacje na temat sekwencji, m.in. takie jak gatunek, długość sekwencji, zawartości poszczególnych nukleotydów itd. Ponadto wykorzystano też język JavaScript.
