Information about the program Sequencer

My project is a program developed as part of the "Python Programming with Elements of Data Analysis and Bioinformatics - Extension" course. It is written in Python using the Django framework and Pandas library. The program renders simple web pages (styled with CSS) in the browser, containing tables generated using the Pandas library. The tables display various information about sequences, such as species, sequence length, nucleotide content, and more. Additionally, JavaScript is also utilized.
